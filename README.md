# pure-js-slideshow
Simple image carousel written with pure JS

### Initialization

```js

// Create an instance
var carousel = new Carousel('carousel');

// Initialize and pass images
carousel.initialize([
    'img/image-1.jpg',
    'img/image-2.jpg',
    'img/image-3.jpg',
    'img/image-4.jpg',
    'img/image-5.jpg'
]);
```
